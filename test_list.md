# Liste de tests à effectuer
Ce document contient la liste de test à effectuer. Deux types de tests seront proposés. Les tests prioritaires sont les tests nécessaires alors que les tests secondaires sont moins importants. 

## Tests prioritaires
Dans tout les tests, il sera important de calculer l'erreur à chaque itération. Puisque le temps sera donné en jour, il sera important que l'erreur soit elle aussi montré en jour.
- Simulation avec les conditions initiales d'*aujourd'hui*. Sommes-nous en mesure de prédire la prochaine apparition?
- Simulation avec les conditions initiales le jour où Clairaut à décider de se lancer dans le projet. Sommes-nous en mesure de prédire comme clairaut son retour?
- Mesurer la période de la comète après une dizaine de rotation.


## Tests secondaires
- Un impact important sur la comète va t'elle la faire décrocher de son orbite? Si on ajoute une composante en vitesse différente.
